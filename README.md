# Voice-Based-Concept-Understanding-Analyser-
An AI web application that evaluates a user's conceptual understanding via audio analysis. It combines OpenAI Whisper speech-to-text, semantic similarity checks, and raw audio quality metrics (like silence percentages). All backend logic is tied to an interactive Streamlit frontend for seamless file uploads and real-time evaluation.
